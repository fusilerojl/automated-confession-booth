Automated Confession Booth

🎓 Academic Project | Bachelor of Science in Computer Engineering

An Arduino-based interactive confession booth designed to improve the waiting experience by integrating motion detection, LCD prompts, LED status indicators, and automated booth management.

Features

- Motion detection using a PIR sensor
- LCD display with guided prompts
- LED status indicators
- Priest notification system
- Automatic timeout handling

Hardware Used

- Arduino Uno
- PIR Motion Sensor
- 16x2 LCD with I2C
- Push Buttons
- LEDs
- Buzzer
